Νέο στη v15:
- Υπενθύμιση πελάτη για προσφορές
- Τηλέφωνο πελάτη και επόμενη ενέργεια
- Αρχείο χωρισμένο σε Προσφορές / Ongoing / Ολοκληρωμένα / Ακυρωμένα
- Όταν επιλέγεται Εγκρίθηκε, αποθηκεύεται ως Σε εξέλιξη / Ongoing
- Κουμπιά Κλήση, Εγκρίθηκε -> Ongoing και Ολοκληρώθηκε στο αρχείο
- Νέο app icon TSITOURAS BELLS

Μετά το upload στο GitHub Pages, ανοίξτε το app μία φορά online για να ανανεωθεί το offline cache.
Για καθαρή εγκατάσταση, σβήστε το παλιό icon από την αρχική οθόνη και κάντε ξανά Add to Home Screen.
